# song-database-
PrevProj of ReSound
HTML을 이용해 노래의 데이터 베이스 역할을 할 수 있는 표 형태의 웹사이트를 만들거야. 사이트의 역할은 사용자에게 노래에 대한 정형화된 여러가지 정보를 물어보고 답변에 따라 노래와 그 정보를 표에 추가하는거야 각 행은 하나의 노래를 의미하고 표의 열은 Rank, Title, Artist, Cover, Producer, Featured Media, Genre, Link로 이루어질거야. 표에서 제일 위의 것을 '항목'이라고 부를게(1행에 있는것) 그리고 그 밑의 것들을(2행 이상의 것) '값'이라고 부를거야. Add Song 버튼을 누르면 Number을 제외한 열의 항목에 대해 질문을 하고 질문이 끝나면 노래 행이 한 줄 추가되어야해. 질문의 형식에 대해 설명할게. 한번에 한개의 질문만을 띄워야하고 next 버튼과 previous 버튼을 통해 질문 간을 이동할 수 있어야해. 또한 질문에 답을 하지 않아도 next 버튼을 통해 넘어갈 수 있어야해. title에 대한 질문은 일본어 제목, 영어 제목, 한국어 제목 순으로 물어봐. cover 가수는 여러 버전이 있으므로 사용자가 여러명을 답변을 할건데 공백으로 넣을 때까지 계속 물어봐야해. genre에 대한 질문은 pop, electronic, rap_hiphop, rock_alt, 80s-90s, metal, rnb, vocaloid, soundtrack, jazz, chill_folk, enka_traditional, miscellaneous 중 복수로 선택할 수 있도록 체크박스로 제시해. link에 대한 질문은 Youtube Link에 대한 질문과 Niconico Link에 대한 질문으로 이루어져있어. link의 답변은 기본적으로 artist에 대응되는거야. 하지만 cover의 답변이 있다면 cover 영상의 link도 답변받아야하기 때문에 link에 대한 질문을 할때는 artist의 영상링크 2개(youtube, niconico), 그리고 각 cover 가수마다 2개씩 질문을 해야해 만약 cover 가수를 입력하지 않았다면 그냥 artist에 대한 링크만 받으면 돼. 표에 나타나는 링크의 '값'은 기본적으로 artist의 youtube link야. 하지만 링크 '항목'을 누르면 youtube link와 niconico link 사이의 변환이 가능해야해. 또한 cover 값이 있는 셀에는 좌우 화살표 버튼이 존재해서 cover 가수를 변환할 수 있어. 다음은 값들의 형식을 알려줄게. 첫번째로 title의 '값'은 기본적으로 표에서 일본어 제목으로 나타나. 그리고 title '항목'을 클릭하면 영어 한국어 순으로 바뀌게 해줘. 한국어에서 다시 클릭하면 일본어 제목이 돼. 또한 title항목의 text를 Title(JP), Title(EN), Title(KR)로 표시해줘. 또한 화살표 버튼으로 cover 가수를 변경할 시에 link의 값이 cover 가수에 대응되는 youtube link로 바뀌고 그 상태에서 link 항목을 누르면 그 cover 가수의 niconico link로 변환이 되어야해. 또한 link 항목의 text를 Youtube Link와 Niconico Link로 각각 표기해줘. 기본적으로는 표를 수정할 수 없지만 edit 버튼을 누르면 셀들을 수정할 수 있어야해. edit 상태에서 genre 열의 값을 우클릭하면 체크박스가 나와서 수정가능하게 만들어. 그리고 genre 항목을 제외한 항목의 값들은 엑셀 입력하듯이 수정가능한 셀로 바뀌어야해. edit 상태를 끄는 버튼을 만들고 각 행의 마지막에 delete할 수 있는 버튼을 만들어줘. 노래 추가 시 노래의 배열 순서는 노래의 영어 제목에 따라 사전순으로 나열하고 영어 제목이 없을경우 제일 아래쪽으로 추가해줘. 노래를 추가할 때 제목 3개 중 어떤 하나라도 기존 노래와 겹친다면 확인 알림을 줘서 추가할지 말지를 결정하게해줘. link 값을 클릭하면 하이퍼링크를 통해 새 탭에서 열리게해줘. 마지막으로 검색기능을 추가해줘. 검색 형식은 '항목:검색어'를 띄어쓰기로 구분해. Rank는 S+ S0 S- A+ A0 A- B+ B0 B- C+ C0 C- 로 구분해. 페이지네이션.플레이리스트. 노래 클릭하면 재생화면. 퀴즈
